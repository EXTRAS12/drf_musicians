# ТЕСТОВОЕ ЗАДАНИЕ

# drf_musicians

Создайте clone:
```
git clone https://github.com/EXTRAS12/drf_musicians.git
```
Перейдите в папку:

```
cd drf_musicians
```
Запустите докер:

```
sudo docker-compose up
```
Перейдите по адресу:
```
http://0.0.0.0:8000/api/v1/swagger/
```
эндпоинт tracklinks соединяет трекс с альбомом и добавляет порядковый номер.
